=======
Reports
=======

The repository keeps cleanup-loop reports in :file:`Documentation/Reports/`.
They are Markdown files because they are written for pull request and release
review, not as long-form rendered TYPO3 manual pages.

Current reports:

*   :file:`typo3-conformance.md` — TYPO3 v14 conventions, extension metadata,
    and compatibility checks.
*   :file:`typo3-security.md` — TYPO3-specific XSS, CSP, iframe, and backend
    hardening review.
*   :file:`typo3-workspaces.md` — workspace overlay and styleguide seeder
    behavior.
*   :file:`typo3-testing.md` — test coverage and CI parity notes.
*   :file:`typo3-docs.md` — documentation freshness against shipped behavior.
*   :file:`security-audit.md` — general dependency, supply chain, and OWASP
    review.
