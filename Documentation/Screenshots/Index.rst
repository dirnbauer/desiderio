===========
Screenshots
===========

These screenshots are captured from the Desiderio styleguide app using the
committed ``b6G5977cw`` shadcn/create preset and shipped Content Block fixture
data. They are useful as a quick visual regression reference for the catalog,
dark mode, and generated preview surfaces.

Styleguide overview
===================

..  figure:: /Images/desiderio-styleguide-overview-light.png
    :alt: Desiderio styleguide overview in light mode.
    :class: with-shadow

    Light mode shows the searchable content element catalog, group navigation,
    badges, and the first generated overview cards for the 255 shipped content
    elements.

..  figure:: /Images/desiderio-styleguide-overview-dark.png
    :alt: Desiderio styleguide overview in dark mode.
    :class: with-shadow

    Dark mode shows the same catalog surface after the theme token switch:
    background, foreground, muted text, borders, badges, and card surfaces all
    come from the preset variables.

Component preview
=================

..  figure:: /Images/desiderio-styleguide-preview-light.png
    :alt: Three Tier Pricing content element preview in light mode.
    :class: with-shadow

    The selected ``Three Tier Pricing`` element shows the component inspector:
    active sidebar entry, preview tab, viewport controls, content type badge,
    and rendered pricing fixture.

..  figure:: /Images/desiderio-styleguide-preview-dark.png
    :alt: Three Tier Pricing content element preview in dark mode.
    :class: with-shadow

    The dark-mode preview checks the same generated pricing UI with dark
    surfaces, muted labels, card borders, and foreground text contrast.
